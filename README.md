# Autoblox Desktop

Graphic programming software for Arduino, ESP32, 
micro:bit and more — powered by Sparky! 🐿️

## Features
- Block-based programming
- Hardware support: Arduino, ESP32, micro:bit, ESP8266
- Built on OpenBlock & Scratch 3.0
